different meta tags

charset attribute: Specifies the character encoding for the document.
name attribute: Assigns a name to the metadata content. It is commonly used for search engine optimization (SEO) and specifying descriptions, keywords
http-equiv attribute: Provides an HTTP header for the information specified. For example, it can be used to refresh or redirect the page.
viewport attribute: Controls the layout and scaling of the viewport on mobile devices.
author attribute: Specifies the author of the document.
keywords attribute: Specifies keywords or phrases that describe the document's content.
robots attribute: Informs web crawlers about how they should index or follow the page
