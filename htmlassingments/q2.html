html doctypes

  • HTML5

    The HTML5 DOCTYPE is the simplest and most commonly used DOCTYPE for HTML5 documents. It triggers standards mode in modern browsers, allowing the use of HTML5 features and elements.

  • XHTML 1.0 Strict

    The XHTML 1.0 Strict DOCTYPE is used for stricter adherence to XHTML 1.0 specifications. It requires closing tags, lowercase element and attribute names, and well-formed XML syntax.

  • XHTML 1.0 Transitional

    The XHTML 1.0 Transitional DOCTYPE allows more flexibility by permitting deprecated elements and attributes from HTML 4.01. It's useful when migrating older HTML code to XHTML.

  • HTML 4.01 Strict

    The HTML 4.01 Strict DOCTYPE is used for strict adherence to HTML 4.01 specifications. It provides a clean, well-structured markup without deprecated elements and attributes.

  • HTML 4.01 Transitional

    The HTML 4.01 Transitional DOCTYPE allows deprecated elements and attributes from older versions of HTML. It's useful for backward compatibility with legacy code.
